Title: Anmol Pandey Portfolio Website
Category: projects

The personal portfolio website of Anmol Pandey is a modern web application designed to showcase his projects, technical skills, and experience in AI and software engineering.

The website is built using React and Vite, which provide a fast development environment and efficient production builds. Styling and UI layout are implemented using Tailwind CSS, allowing for responsive and modern component design.

The site includes multiple interactive UI components such as animated hero sections, orbiting skill visualizations, and dynamic project showcases.

Animations and motion effects are implemented using Motion (Framer Motion compatible API) to create smooth transitions and engaging user interactions.

The site also integrates an AI-powered chatbot assistant that allows visitors to ask questions about Anmol's background, projects, and technical expertise.

The chatbot uses a Retrieval-Augmented Generation (RAG) backend powered by:

- FastAPI
- FAISS vector database
- Sentence Transformers for embeddings
- Google Gemini LLM for response generation

The portfolio is deployed as a static frontend application and communicates with the AI chatbot backend through an API endpoint.

Main technologies used in the portfolio website:

Frontend:
- React
- Vite
- Tailwind CSS
- Motion / Framer Motion
- React Router

Visualization & UI:
- Cobe (interactive globe visualization)
- Lucide icons

Backend AI assistant:
- Python
- FastAPI
- FAISS
- Sentence Transformers
- LangChain
- Google Gemini LLM

The portfolio demonstrates Anmol Pandey's ability to design modern web applications, integrate AI systems, and build interactive developer-focused user experiences.
